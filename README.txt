ETL PIPELINE
============

HOW TO USE:
1. Put your Excel files in input_files/
2. Run: python code/load.py
3. Check output_files/ for results.xlsx

STRUCTURE:
- code/         → Python scripts (Extract, Transform, Load)
- input_files/  → Your Excel data goes here
- output_files/ → Processed results appear here
- venv/         → Python environment (ignore this)
